## Report Format Standards

All reports in this folder follow:

- **Thynaptic Technical Framework Report Template:** Overview, Architectural Context, Methodology, Evaluation Results, System Behavior Analysis, Limitations, Forward Research Trajectories
- **Standard Front Matter:** Title, TR-YYYY-##, Authors, Division, Date, Version
- **Evidence-Grounded Claims:** All architectural and performance claims supported by evaluation data
- **Measurable Metrics:** Accuracy, latency, routing distribution, cognitive performance observations
